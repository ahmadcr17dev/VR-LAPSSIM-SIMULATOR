# VR Laparoscopic Appendectomy Simulator

A Virtual Reality (VR) surgical training simulator designed to provide an immersive and interactive environment for practicing **Laparoscopic Appendectomy** procedures. This project was developed as a Final Year Project (FYP) using **Unity**, **C#**, **Blender**, and VR technologies to help students and medical trainees gain hands-on experience in a safe, risk-free environment.

---

## 📌 Overview

The VR Laparoscopic Appendectomy Simulator replicates the workflow of a real laparoscopic appendectomy procedure. Users interact with virtual surgical instruments inside a simulated operating room, allowing them to practice essential surgical skills without the risks associated with real patients.

The simulator emphasizes realistic interaction, anatomical visualization, and procedural learning through immersive virtual reality.

---

## 🎯 Objectives

* Develop an immersive VR-based surgical training environment.
* Simulate the complete laparoscopic appendectomy procedure.
* Improve hand-eye coordination and instrument handling.
* Provide a safe and repeatable learning platform for surgical practice.
* Reduce dependence on expensive physical training equipment.

## 🚀 Getting Started

### Prerequisites

* Unity Editor (recommended version used for the project)
* Visual Studio
* VR Headset compatible with Unity XR
* Git

### Installation

1. Clone the repository.

```bash
git clone https://github.com/ahmadcr17dev/VR-LAPSSIM-SIMULATOR.git
```

2. Open the project using Unity Hub.

3. Allow Unity to import all required packages.

4. Connect a supported VR headset.

5. Press **Play** to run the simulator.

---

## 🎮 How It Works

The simulator guides users through the major stages of a laparoscopic appendectomy. Users manipulate virtual laparoscopic instruments using VR controllers to perform surgical tasks within a realistic operating environment. The experience is designed to enhance procedural understanding, precision, and familiarity with minimally invasive surgical techniques.
